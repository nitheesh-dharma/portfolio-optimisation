# Portfolio Optimisation Using Modern Portfolio Theory

## Overview

This project applies Modern Portfolio Theory to construct and analyse optimal investment portfolios using historical asset return data. The project explores diversification, covariance structures, efficient portfolios, and risk-adjusted performance.

## Objectives

- Analyse historical asset returns
- Construct portfolios using Modern Portfolio Theory
- Simulate random portfolio allocations
- Generate the efficient frontier
- Identify minimum variance portfolios
- Maximise risk-adjusted returns using the Sharpe ratio
- Explore diversification and portfolio risk


## Features

- Historical asset return analysis
- Covariance matrix estimation
- Random portfolio simulation
- Efficient frontier visualisation
- Maximum Sharpe ratio portfolio optimisation
- Minimum variance portfolio construction
- Risk-return trade-off analysis

## Technologies Used

- Python
- NumPy
- Pandas
- SciPy
- Matplotlib
- yfinance
- Jupyter Notebook

## Key Findings

- Diversification reduces portfolio risk
- Efficient portfolios maximise return for a given level of risk
- The Sharpe ratio provides a useful measure of risk-adjusted performance
- Asset covariance plays a critical role in portfolio construction
